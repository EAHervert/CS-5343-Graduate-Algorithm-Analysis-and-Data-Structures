# CS 5343 — Algorithms & Data Structures

Living DSA notebook — reference implementations, worked problems, and study notes.
Originated as coursework for **CS 5343 — Graduate Algorithm Analysis & Data Structures**
(UT Dallas, Summer 2019 — textbook: Goodrich, Tamassia & Mount, *Data Structures and Algorithms in C++*),
rebuilt as an interview-prep and teaching resource.
C++ is the primary language.

## Original course scope

1. Asymptotic notation, recurrences, algorithm analysis
2. Lists, stacks, queues, hashing, priority queues
3. Binary search trees, balanced BSTs, B-trees
4. Graphs, DFS, topological ordering
5. BFS, Dijkstra's algorithm
6. Prim's & Kruskal's algorithms, Disjoint-Set Union-Find
7. Algorithm design techniques
